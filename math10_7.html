Coordinate Geometry

Coordinate Geometry Overview

Coordinate geometry, also known as analytic geometry, is the study of geometry using a coordinate system. It involves using algebraic methods to solve geometric problems. The central idea is to represent geometric shapes, points, and objects in a coordinate plane (usually Cartesian coordinates) and to apply algebraic techniques to analyze and solve geometric problems.

In coordinate geometry, every point in the plane is represented by an ordered pair of numbers (x, y), where x is the horizontal coordinate and y is the vertical coordinate.

Key Concepts:

  • Coordinate Plane: The coordinate plane consists of two perpendicular axes: the x-axis (horizontal) and the y-axis (vertical). The point where the two axes intersect is the origin, denoted by (0, 0).
  • The plane is divided into four quadrants:
    • Quadrant I: x > 0 and y > 0
    • Quadrant II: x < 0 and y > 0
    • Quadrant III: x < 0 and y < 0
    • Quadrant IV: x > 0 and y < 0

Important Formulas:

1. Distance Formula:
d = √((x₂ - x₁)² + (y₂ - y₁)²)

2. Midpoint Formula:
M = ((x₁ + x₂)/2, (y₁ + y₂)/2)

3. Slope Formula:
m = (y₂ - y₁) / (x₂ - x₁)

4. Equation of a Line (Point-Slope Form):
y - y₁ = m(x - x₁)

5. Equation of a Line (Slope-Intercept Form):
y = mx + b

6. Equation of a Line (General Form):
Ax + By + C = 0

7. Circle Equation:
(x - h)² + (y - k)² = r²

Example Problems:

1. Distance Between Two Points:

Find the distance between points A(2, 3) and B(5, 7).

Using the distance formula:
d = √((5 - 2)² + (7 - 3)²) = √(3² + 4²) = √(9 + 16) = √25 = 5

Therefore, the distance between A and B is 5 units.

2. Midpoint of Two Points:

Find the midpoint of points A(2, 3) and B(5, 7).

Using the midpoint formula:
M = ((2 + 5)/2, (3 + 7)/2) = (7/2, 10/2) = (3.5, 5)

Therefore, the midpoint of A and B is (3.5, 5).

3. Equation of a Line:

Find the equation of the line passing through the points A(1, 2) and B(3, 6).

First, find the slope using the slope formula:
m = (6 - 2) / (3 - 1) = 4 / 2 = 2

Now, using the point-slope form, the equation of the line is:
y - 2 = 2(x - 1)
Simplifying:
y - 2 = 2x - 2 → y = 2x

Therefore, the equation of the line is y = 2x.
