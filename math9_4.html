Solving Inequalities

Solving Inequalities Deeply

An inequality is a mathematical expression that relates two values that may not be equal. It is similar to an equation, but instead of using an equal sign (=), we use inequality symbols to show that one side of the expression is larger or smaller than the other side.

1. Types of Inequalities

Inequalities are divided into different types based on the symbol used:

  • Strict inequalities:
    • < (less than)
    • > (greater than)
  • Non-strict inequalities:
    • ≤ (less than or equal to)
    • ≥ (greater than or equal to)

Each of these symbols represents a range of values that satisfy the inequality.

2. Solving Linear Inequalities

A linear inequality involves a linear expression, i.e., the highest degree of the variable is 1. The basic rules for solving linear inequalities are similar to solving equations, but we must take special care when multiplying or dividing by negative numbers.

Steps for Solving Linear Inequalities

  1. Isolate the variable on one side of the inequality (similar to solving an equation).
  2. Simplify both sides if needed (combine like terms).
  3. If you multiply or divide by a negative number, you must reverse the inequality sign.
  4. Graph the solution on a number line (if required).

Example 1:

Solve: 2x - 3 ≥ 7

Step 1: Add 3 to both sides:

2x ≥ 10

Step 2: Divide both sides by 2:

x ≥ 5

The solution is x ≥ 5.

Example 2:

Solve: -3x < 9

Step 1: Divide both sides by -3 (remember to reverse the inequality sign):

x > -3

The solution is x > -3.

3. Solving Compound Inequalities

Compound inequalities involve two or more inequalities that are joined together by the words "and" or "or". The solution is found by solving each inequality separately and then combining the results.

"And" Compound Inequality:

Example: Solve 3 ≤ x + 2 ≤ 5

Step 1: Solve the left inequality: 3 ≤ x + 2 ⟶ x ≥ 1

Step 2: Solve the right inequality: x + 2 ≤ 5 ⟶ x ≤ 3

Step 3: Combine both parts: The solution is 1 ≤ x ≤ 3.

"Or" Compound Inequality:

Example: Solve x < -2 or x ≥ 4

The solution is the union of both inequalities: x < -2 or x ≥ 4.

4. Solving Absolute Value Inequalities

Absolute value inequalities involve expressions that have absolute value bars around the variable or expression. The absolute value represents the distance of a number from zero on the number line, regardless of direction.

For inequalities of the form |x| < a:

The solution is -a < x < a.

For inequalities of the form |x| > a:

The solution is x < -a or x > a.

Example 1:

Solve: |x - 3| < 5

-5 < x - 3 < 5

Add 3 to all parts: -2 < x < 8

The solution is -2 < x < 8.

Example 2:

Solve: |x + 4| ≥ 7

This gives two inequalities:

  • x + 4 ≥ 7 ⟶ x ≥ 3
  • x + 4 ≤ -7 ⟶ x ≤ -11

The solution is x ≤ -11 or x ≥ 3.

5. Graphing Inequalities

When graphing inequalities on a number line:

  • For strict inequalities (e.g., x > 3), use an open circle at the point.
  • For non-strict inequalities (e.g., x ≥ 3), use a closed circle at the point.
  • Use arrows to indicate the direction of the solution (to the right for greater values and to the left for smaller values).

Formulas Related to Solving Inequalities

  • Addition and Subtraction Rule: If a ≤ b, then a + c ≤ b + c and a - c ≤ b - c, for any real number c.
  • Multiplication and Division Rule (by a positive number): If a ≤ b, then a × c ≤ b × c and a ÷ c ≤ b ÷ c, for any positive real number c.
  • Multiplication and Division Rule (by a negative number): If a ≤ b, then a × c ≥ b × c and a ÷ c ≥ b ÷ c, for any negative real number c (note: the inequality flips when multiplying or dividing by negative numbers).

6. Special Tips for Solving Inequalities:

  • Always be careful when multiplying or dividing by negative numbers, as the inequality sign will flip.
  • Graph the solution on a number line for a visual representation.
  • Check the boundary points (if any) to ensure they satisfy the inequality.
