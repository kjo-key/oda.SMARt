Vectors in Two Dimensions

1. Definition of a Vector

A vector is a mathematical entity that has both **magnitude** and **direction**.

2. Representation of Vectors

Vectors in two-dimensional space can be written in different ways:

Column Form:

A = [ x , y ]

Unit Vector Notation:

A = xi + yj

3. Magnitude of a Vector

The length or magnitude of a vector **A = (x, y)** is calculated as:

|A| = √(x² + y²)

4. Direction (Angle) of a Vector

The direction of a vector with respect to the x-axis is given by:

θ = tan⁻¹(y / x)

5. Vector Addition

If A = (x₁, y₁) and B = (x₂, y₂), then:

A + B = (x₁ + x₂, y₁ + y₂)

6. Vector Subtraction

A - B = (x₁ - x₂, y₁ - y₂)

7. Scalar Multiplication

If k is a scalar and A = (x, y), then:

kA = (kx, ky)

8. Dot Product of Two Vectors

The dot product of two vectors A and B is calculated as:

A • B = x₁x₂ + y₁y₂

or equivalently,

A • B = |A| |B| cosθ

9. Cross Product in 2D

In two dimensions, the cross product is given by:

A × B = (x₁y₂ - y₁x₂)

10. Unit Vector

A unit vector in the direction of A is given by:

Unit vector Â = A / |A|

11. Projection of a Vector

The projection of vector A onto vector B is given by:

ProjB A = (A • B) / |B|² * B

12. Vector Rotation

A vector can be rotated by an angle θ using:

A' = (x cosθ - y sinθ, x sinθ + y cosθ)

13. Vector Components

A vector A at an angle θ can be broken into components:

Ax = |A| cosθ

Ay = |A| sinθ

14. Position Vector

The position vector of a point (x, y) from the origin is:

R = xi + yj

15. Velocity and Acceleration Vectors

For a moving object with position vector R(t):

Velocity: V = dR/dt

Acceleration: A = d²R/dt²

16. Work Done by a Force

Work W done by a force F moving an object along displacement d:

W = F • d = |F| |d| cosθ

17. Torque

Torque τ produced by a force F at a position vector r:

τ = r × F

18. Vector Equation of a Line

The equation of a line passing through A with direction B:

r = A + tB
