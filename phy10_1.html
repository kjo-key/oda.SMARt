Vector Quantities: A Deep Dive

1. What Are Vector Quantities?

Vectors have both magnitude and direction. They are different from scalar quantities, which only have magnitude.

2. Examples of Vector Quantities

  • Displacement ( Δx )
  • Velocity ( v = Δx / Δt )
  • Acceleration ( a = Δv / Δt )
  • Force ( F = ma )
  • Momentum ( p = mv )

3. Representation of a Vector

A = Ax î + Ay ĵ + Az k̂

Alternative notation:

A = (Ax, Ay, Az)

4. Magnitude and Direction

Magnitude (Length of a Vector):

|A| = √(Ax² + Ay²)   (for 2D)
|A| = √(Ax² + Ay² + Az²)   (for 3D)

Direction (Angle θ):

θ = tan⁻¹(Ay / Ax)

5. Vector Operations

1. Vector Addition

Graphical Method (Head-to-Tail Rule)

Component Method:

R = A + B = (Ax + Bx, Ay + By)

2. Vector Subtraction

A - B = (Ax - Bx, Ay - By)

3. Scalar (Dot) Product

Produces a scalar value:

A · B = |A| |B| cosθ

4. Vector (Cross) Product

Produces a new vector:

A × B = |A| |B| sinθ n̂

5. Projection of a Vector

Proj_A B = (A · B) / |A|

6. Unit Vector

Â = A / |A|

7. Applications of Vectors in Physics

  • Newton’s Laws: F = ma
  • Work Done: W = F · d = Fd cosθ
  • Torque: τ = r × F
  • Electric Fields: E = F / q
  • Magnetic Force: F = q (v × B)

8. Vector Decomposition

Any vector can be broken into perpendicular components.

Ax = A cosθ
Ay = A sinθ

9. Vectors in Different Coordinate Systems

Cartesian Coordinates

A = (Ax, Ay, Az)

Polar Coordinates

A = (r, θ)
Ax = r cosθ
Ay = r sinθ

10. Summary

  • Vectors have magnitude and direction.
  • Vector operations include addition, subtraction, dot product, and cross product.
  • Unit vectors help in standardizing directions.
  • Vectors are fundamental in mechanics, electromagnetism, and AI simulations.
