Set Theory Concepts

1. Definition of a Set

A set is a collection of distinct objects, considered as an object in its own right. These objects are called elements or members of the set.

Example: A = {1, 2, 3, 4, 5, 6, 7, 8, 9}

2. Types of Sets

  • Finite Set: A set that has a definite number of elements. Example: A = {1, 2, 3}
  • Infinite Set: A set that has an infinite number of elements. Example: B = {1, 2, 3, 4, 5, ...}
  • Equal Set: Two sets that contain exactly the same elements. Example: A = {1, 2, 3} and B = {1, 2, 3} are equal sets.
  • Null Set: A set with no elements. Example: C = {} or C = ∅
  • Singleton Set: A set with only one element. Example: D = {5}
  • Universal Set: The set that contains all elements under consideration for a particular discussion. It is often denoted by U.
  • Subset: A set A is a subset of set B (denoted A ⊆ B) if every element of A is also an element of B.

3. Set Operations

Union (A ∪ B)

The union of two sets A and B is the set of elements that are in A, in B, or in both.

Formula:
A ∪ B = {x | x ∈ A or x ∈ B}
If |A| and |B| are the number of elements in sets A and B, and |A ∩ B| is the number of elements in the intersection, then:
|A ∪ B| = |A| + |B| - |A ∩ B|

Intersection (A ∩ B)

The intersection of two sets A and B is the set of elements that are common to both A and B.

Formula:
A ∩ B = {x | x ∈ A and x ∈ B}

Difference (A - B)

The difference of two sets A and B (also called the complement of B in A) is the set of elements that are in A but not in B.

Formula:
A - B = {x | x ∈ A and x ∉ B}

Complement (A')

The complement of a set A (denoted A') consists of all elements in the universal set U that are not in A.

Formula:
A' = {x | x ∉ A}

Symmetric Difference (A Δ B)

The symmetric difference of two sets A and B is the set of elements that are in either A or B, but not in both.

Formula:
A Δ B = (A - B) ∪ (B - A)

4. Venn Diagrams

Venn diagrams are graphical representations of sets and their relationships. They show all possible logical relations between a finite collection of sets. The universal set is usually represented by a rectangle, while subsets are shown as circles within it.

5. Cardinality of a Set

The cardinality of a set A, denoted |A|, is the number of elements in A. If a set is finite, its cardinality is simply the count of elements. For infinite sets, the cardinality represents the type or size of infinity.

  • Finite Set: |A| = the number of elements in A
  • Infinite Set: Cardinality is often described by infinite numbers such as ℵ₀ for countably infinite sets.

6. Power Set

The power set of a set A, denoted P(A), is the set of all subsets of A. The power set of a set with n elements has 2ⁿ subsets.

Formula:
P(A) = {B | B ⊆ A}

7. Cartesian Product (A × B)

The Cartesian product of two sets A and B, denoted A × B, is the set of ordered pairs where the first element is from A and the second is from B.

Formula:
A × B = {(a, b) | a ∈ A and b ∈ B}

8. Laws of Set Theory

  • Commutative Laws:
    • A ∪ B = B ∪ A
    • A ∩ B = B ∩ A
  • Associative Laws:
    • (A ∪ B) ∪ C = A ∪ (B ∪ C)
    • (A ∩ B) ∩ C = A ∩ (B ∩ C)
  • Distributive Laws:
    • A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)
    • A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)

9. Set Identities

  • De Morgan’s Laws:
    • (A ∪ B)' = A' ∩ B'
    • (A ∩ B)' = A' ∪ B'
  • Double Complement Law: (A')' = A
  • Absorption Laws:
    • A ∪ (A ∩ B) = A
    • A ∩ (A ∪ B) = A

10. Multiset

A multiset allows for multiple occurrences of the same element. Unlike sets, where elements are distinct, multisets can have repeated elements.

Example of a multiset: {1, 1, 2, 3}
