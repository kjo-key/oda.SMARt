‚úÖ 2. Python Syntax, Variables & Data Types

2.1 Python Syntax

Python uses indentation instead of curly braces {}:

# Correct indentation
if 5 > 2:
    print("5 is greater than 2")

# Incorrect indentation (will cause an error)
if 5 > 2:
print("5 is greater than 2")  # Error: No indentation

2.2 Variables in Python

Variables store data in memory.

Rules for Variable Names:

  • Can contain letters, numbers, and underscores (_).
  • Cannot start with a number.
  • Are case-sensitive (name and Name are different).

Example:

name = "Yoseph"
age = 16
height = 1.75  # meters
is_student = True  # Boolean

print(name, age, height, is_student)

2.3 Data Types in Python

Python has several built-in data types:

‚úÖ 3. Input & Output Operations

3.1 Output with print()

The print() function displays output:

print("Hello, World!")
print(10 + 5)  # Output: 15

3.2 Formatting Output

String formatting allows you to include variables in your output:

name = "Yoseph"
age = 16
print(f"My name is {name} and I am {age} years old.")  # f-string

Other formatting methods:

# Using .format()
print("My name is {} and I am {} years old.".format(name, age))

# Using % formatting
print("My name is %s and I am %d years old." % (name, age))

3.3 Input from the User

You can take input from the user using the input() function:

name = input("Enter your name: ")
age = int(input("Enter your age: "))  # Convert input to integer
print(f"Hello, {name}! You are {age} years old.")

Control Flow & Functions

1. Conditional Statements (if, elif, else)

Conditional statements are used to make decisions based on conditions.

# Example:
x = 10
if x > 5:
    print("x is greater than 5")
elif x == 5:
    print("x is equal to 5")
else:
    print("x is less than 5")

2. Loops (for, while)

Loops allow you to repeat a block of code multiple times.

For Loop: Used for iterating over a sequence (like a list, string, or range).

# Example:
for i in range(5):
    print(i)

While Loop: Runs as long as a condition is true.

# Example:
x = 0
while x < 5:
    print(x)
    x += 1

3. Functions (def, arguments, return values)

Functions allow you to group code into reusable blocks.

# Example:
def add(a, b):
    return a + b

result = add(3, 4)
print(result)  # Output: 7

4. Recursion

Recursion occurs when a function calls itself.

# Example: Factorial function using recursion
def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)

print(factorial(5))  # Output: 120

Data Structures & String Manipulation

1. Data Structures

Lists

What is a List?

A list is a collection of items that are ordered and changeable.

# Example:
my_list = [1, 2, 3, "apple"]

Tuples

What is a Tuple?

A tuple is similar to a list, but it is immutable.

# Example:
my_tuple = (1, 2, 3, "apple")

Sets

What is a Set?

A set is a collection of unique items.

# Example:
my_set = {1, 2, 3, "apple"}

Dictionaries

What is a Dictionary?

A dictionary is a collection of key-value pairs.

# Example:
my_dict = {"name": "Alice", "age": 25}

2. String Manipulation

What is a String?

A string is a sequence of characters.

# Example:
my_string = "Hello, World!"

3. List Comprehension

What is List Comprehension?

List comprehension is a way to create a new list by applying an expression to each item in an existing list.

# Example:
squares = [x**2 for x in range(10)]
print(squares)  # Output: [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

Classes & Objects

1. Creating Classes & Objects

A class is a blueprint for creating objects.

# Example:
class Car:
    pass

my_car = Car()

2. Constructors (__init__ method)

A constructor is a special method called __init__.

# Example:
class Car:
    def __init__(self, brand, model, year):
        self.brand = brand
        self.model = model
        self.year = year

my_car = Car("Toyota", "Corolla", 2020)

3. Instance & Class Variables

Instance Variables: Belong to a specific instance.

Class Variables: Shared by all instances.

# Example:
class Car:
    wheels = 4  # Class variable

    def __init__(self, brand, model, year):
        self.brand = brand  # Instance variable
        self.model = model
        self.year = year

car1 = Car("Toyota", "Corolla", 2020)
car2 = Car("Honda", "Civic", 2021)

print(car1.wheels)  # Output: 4
print(car2.wheels)  # Output: 4

4. Methods

Methods define the behaviors of objects.

# Example:
class Car:
    def __init__(self, brand, model, year):
        self.brand = brand
        self.model = model
        self.year = year

    def display_info(self):
        print(f"{self.brand} {self.model}, {self.year}")

my_car = Car("Toyota", "Corolla", 2020)
my_car.display_info()  # Output: Toyota Corolla, 2020

5. Inheritance

Inheritance allows a class to inherit attributes and methods from another class.

# Example:
class Vehicle:
    def __init__(self, brand, model):
        self.brand = brand
        self.model = model

class Car(Vehicle):
    def __init__(self, brand, model, year):
        super().__init__(brand, model)
        self.year = year

my_car = Car("Honda", "Civic", 2022)
print(my_car.brand)  # Output: Honda
