Learn HTML - Complete Guide for beginners

1. What is HTML?

HTML stands for HyperText Markup Language. It is the standard language for creating web pages. HTML uses tags to structure content, such as headings, paragraphs, images, and links. It is not a programming language; it is a markup language used to define the structure of web content.

2. Basic Structure of an HTML Document

Every HTML document follows a basic structure. Here‚Äôs an example:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My First Web Page</title>
</head>
<body>
  <h1>Welcome to My Website</h1>
  <p>This is a paragraph of text.</p>
</body>
</html>

Explanation:

  • <!DOCTYPE html>: Declares the document type and version of HTML (HTML5 in this case).
  • <html>: The root element of the HTML document.
  • <head>: Contains meta-information about the document, such as the title and character set.
  • <title>: Sets the title of the webpage (shown in the browser tab).
  • <body>: Contains the visible content of the webpage.

3. HTML Elements and Tags

Elements: The building blocks of HTML. They define the structure and content of a webpage.

Tags: Used to define elements. Tags are enclosed in angle brackets (< >).

Example:

<p>This is a paragraph.</p>
  • <p> is the opening tag.
  • </p> is the closing tag.
  • The content between the tags is the element‚Äôs content.

4. Common HTML Tags

Headings (<h1> to <h6>)

Headings are used to define titles and subtitles. <h1> is the largest, and <h6> is the smallest.

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Smaller Subheading</h3>

Paragraphs (<p>)

Used to define paragraphs of text.

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

Links (<a>)

Used to create hyperlinks. The href attribute specifies the link‚Äôs destination.

<a href="https://www.example.com">Visit Example</a>

Images (<img>)

Used to embed images. The src attribute specifies the image path, and alt provides alternative text.

<img src="image.jpg" alt="Description of the image">

Lists

Unordered List (<ul>):

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

Ordered List (<ol>):

<ol>
  <li>First Item</li>
  <li>Second Item</li>
  <li>Third Item</li>
</ol>

Tables (<table>)

Used to create tables.

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>25</td>
  </tr>
  <tr>
    <td>Jane</td>
    <td>30</td>
  </tr>
</table>

Forms (<form>)

Used to collect user input.

<form>
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <br>
  <label for="email">Email:</label>
  <input type="email" id="email" name="email">
  <br>
  <button type="submit">Submit</button>
</form>

5. Attributes

Attributes provide additional information about HTML elements. They are placed inside the opening tag.

Example:

<a href="https://www.example.com" target="_blank">Visit Example</a>
  • href: Specifies the link‚Äôs destination.
  • target="_blank": Opens the link in a new tab.

6. Semantic HTML

Semantic HTML uses meaningful tags to describe the content‚Äôs purpose. Examples include:

  • <header>: Defines the header of a page or section.
  • <nav>: Defines navigation links.
  • <main>: Defines the main content of a page.
  • <footer>: Defines the footer of a page or section.

Example:

<header>
  <h1>Welcome to My Website</h1>
  <nav>
    <a href="#home">Home</a>
    <a href="#about">About</a>
  </nav>
</header>
<main>
  <p>This is the main content.</p>
</main>
<footer>
  <p>© 2024 My Website</p>
</footer>

Advanced HTML Guide

1. HTML5 Semantic Elements

Semantic elements describe the meaning of the content they contain. They improve accessibility, SEO, and make your code easier to read.

Key Semantic Elements:

  • <header>: Represents the header of a page or section.
  • <nav>: Defines navigation links.
  • <main>: Contains the main content of the page.
  • <section>: Defines a section of related content.
  • <article>: Represents independent, self-contained content (e.g., blog post).
  • <aside>: Contains content related to the main content (e.g., sidebar).
  • <footer>: Represents the footer of a page or section.

Example:

<header>
  <h1>My Website</h1>
  <nav>
    <a href="#home">Home</a>
    <a href="#about">About</a>
  </nav>
</header>
<main>
  <article>
    <h2>Blog Post Title</h2>
    <p>This is a blog post.</p>
  </article>
  <aside>
    <h3>Related Links</h3>
    <ul>
      <li><a href="#link1">Link 1</a></li>
      <li><a href="#link2">Link 2</a></li>
    </ul>
  </aside>
</main>
<footer>
  <p>© 2024 My Website</p>
</footer>

2. Forms and Input Types

HTML5 introduced new input types and attributes for better user experience and validation.

Common Input Types:

  • text: Single-line text input.
  • email: Email address input.
  • password: Password input (hidden characters).
  • number: Numeric input.
  • date: Date picker.
  • file: File upload.
  • checkbox: Checkbox input.
  • radio: Radio button input.
  • submit: Submit button.

Example:

<form>
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" required>
  <br>
  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required>
  <br>
  <label for="password">Password:</label>
  <input type="password" id="password" name="password" required>
  <br>
  <label for="age">Age:</label>
  <input type="number" id="age" name="age" min="18" max="100">
  <br>
  <label for="dob">Date of Birth:</label>
  <input type="date" id="dob" name="dob">
  <br>
  <button type="submit">Submit</button>
</form>

Example:

  width="600"

5. Accessibility (ARIA Roles and Attributes)

ARIA (Accessible Rich Internet Applications) helps make web content more accessible to people with disabilities.

Common ARIA Attributes:

  • role: Defines the purpose of an element (e.g., role="navigation").
  • aria-label: Provides a label for an element.
  • aria-labelledby: Associates an element with a label.
  • aria-describedby: Provides a description for an element.

Example:

<nav role="navigation" aria-label="Main Navigation">
  <ul>
    <li><a href="#home">Home</a></li>
    <li><a href="#about">About</a></li>
  </ul>
</nav>

6. Meta Tags for SEO and Social Media

Meta tags provide metadata about your webpage, which is used by search engines and social media platforms.

Common Meta Tags:

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta name="description" content="A description of your webpage">
  <meta name="keywords" content="HTML, CSS, JavaScript">
  <meta name="author" content="Your Name">
  <!-- Open Graph Meta Tags for Social Media -->
  <meta property="og:title" content="Your Webpage Title">
  <meta property="og:description" content="A description of your webpage">
  <meta property="og:image" content="image.jpg">
  <meta property="og:url" content="https://www.example.com">
</head>

7. HTML Entities and Special Characters

Use HTML entities to display special characters.

Common Entities:

  • &lt;: <
  • &gt;: >
  • &amp;: &
  • &copy;: ©
  • &nbsp;: Non-breaking space

Example:

<p>This is a &lt;div&gt; element.</p>
<p>Copyright &copy; 2024 My Website.</p>

8. Data Attributes

Data attributes (data-*) allow you to store custom data in HTML elements.

Example:

<div data-user-id="123" data-role="admin">User Info</div>

Accessing Data Attributes in JavaScript:

<script>
  const userDiv = document.querySelector('div');
  console.log(userDiv.dataset.userId); // Output: 123
</script>

Use the W3C Validator:

Complete CSS Guide

1. What is CSS?

CSS stands for Cascading Style Sheets. It is used to style HTML elements, such as colors, fonts, spacing, and layout. CSS works by selecting HTML elements and applying styles to them.

2. CSS Syntax

CSS consists of selectors and declarations.

Syntax:

selector {
  property: value;
}

Example:

h1 {
  color: blue;
  font-size: 24px;
}

3. Selectors

Selectors are used to target HTML elements for styling.

Common Selectors:

  • Element Selector: Targets all instances of an element. 
    p {
  color: red;
}
  • Class Selector: Targets elements with a specific class. 
    .text-center {
  text-align: center;
}
  • ID Selector: Targets a single element with a specific ID. 
    #header {
  background-color: yellow;
}
  • Attribute Selector: Targets elements with a specific attribute. 
    input[type="text"] {
  border: 1px solid #ccc;
}

4. Box Model

Every element in CSS is a box, consisting of:

  • Content: The actual content (text, image, etc.).
  • Padding: Space between the content and the border.
  • Border: A line around the padding and content.
  • Margin: Space outside the border.

Example:

.box {
  width: 200px;
  padding: 20px;
  border: 2px solid black;
  margin: 10px;
}

5. Display and Positioning

Display Properties:

  • block: Takes the full width and starts on a new line.
  • inline: Takes only the necessary width and stays in the same line.
  • inline-block: Combines features of both block and inline.
  • none: Hides the element.

Positioning:

  • static: Default positioning.
  • relative: Positioned relative to its normal position.
  • absolute: Positioned relative to the nearest positioned ancestor.
  • fixed: Positioned relative to the viewport.
  • sticky: Toggles between relative and fixed.

Example:

.box {
  position: relative;
  top: 20px;
  left: 30px;
}

6. Flexbox

Flexbox is a layout model for arranging items in a container.

Key Properties:

  • display: flex: Enables flexbox.
  • flex-direction: Defines the direction of items (row, column).
  • justify-content: Aligns items along the main axis.
  • align-items: Aligns items along the cross axis.

Example:

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;

JavaScript Beginner's Guide

What is JavaScript?

JavaScript is a programming language used to make web pages interactive. It runs in your browser and allows you to manipulate HTML, CSS, and data dynamically. It‚Äôs also used on the server-side (with Node.js) and in mobile app development.

Setting Up JavaScript

You don‚Äôt need any special tools to start coding in JavaScript. All you need is:

  1. A browser (like Chrome, Firefox, or Edge).
  2. A text editor (like VS Code, Sublime Text, or Notepad).

You can write JavaScript directly in your browser‚Äôs console or in an HTML file.

Writing Your First JavaScript Code

Let‚Äôs start with a simple example. Open your browser‚Äôs developer tools (press F12 or Ctrl+Shift+I), go to the Console tab, and type: 

console.log("Hello, World!");

Press Enter, and you‚Äôll see Hello, World! printed in the console. Congratulations! You‚Äôve just written your first JavaScript program.

JavaScript Basics

1. Variables

Variables are used to store data. In JavaScript, you can declare variables using:

  • let: For variables that can change.
  • const: For variables that cannot change (constants).
  • var: Older way (avoid in modern JavaScript).

Example:

let name = "Alice";
const age = 25;
var city = "New York"; // Avoid using var
console.log(name, age, city);

2. Data Types

JavaScript has several data types:

  • Primitive Types: string, number, boolean, null, undefined, symbol, bigint.
  • Objects: object, array, function.

Example:

let name = "Alice"; // string
let age = 25; // number
let isStudent = true; // boolean
let car = null; // null
let job; // undefined

3. Operators

- Arithmetic: +, -, *, /, %, ** (exponentiation).
- Comparison: ==, ===, !=, !==, >, <, >=, <=.
- Logical: &&, ||, !.

Example:

let a = 10;
let b = 5;
console.log(a + b); // 15
console.log(a > b); // true
console.log(a === 10 && b === 5); // true

4. Conditionals

Use if, else if, and else to make decisions.

Example:

let age = 18;
if (age >= 18) {
  console.log("You are an adult.");
} else {
  console.log("You are a minor.");
}

5. Loops

Loops are used to repeat code. Common loops are for, while, and do...while.

Example:

for (let i = 0; i < 5; i++) {
  console.log(i); // Prints 0, 1, 2, 3, 4
}

let i = 0;
while (i < 5) {
  console.log(i); // Prints 0, 1, 2, 3, 4
  i++;
}

6. Functions

Functions are reusable blocks of code. You can define them using the function keyword or arrow functions (=>).

Example:

function greet(name) {
  console.log("Hello, " + name);
}
greet("Alice"); // Hello, Alice

// Arrow function
const greetArrow = (name) => {
  console.log("Hello, " + name);
};
greetArrow("Bob"); // Hello, Bob

7. Arrays

Arrays are used to store multiple values in a single variable.

Example:

let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits[0]); // Apple
fruits.push("Orange"); // Add to the end
console.log(fruits); // ["Apple", "Banana", "Cherry", "Orange"]

8. Objects

Objects are collections of key-value pairs.

Example:

let person = {
  name: "Alice",
  age: 25,
  isStudent: true,
};
console.log(person.name); // Alice
person.city = "New York"; // Add a new property
console.log(person); // { name: "Alice", age: 25, isStudent: true, city: "New York" }

Intermediate Concepts

1. DOM Manipulation

The Document Object Model (DOM) is how JavaScript interacts with HTML and CSS. You can use JavaScript to change the content, style, and structure of a webpage.

Example:

<!DOCTYPE html>
<html>
  <body>
    <h1 id="title">Hello, World!</h1>
    <button onclick="changeText()">Click Me</button>
    <script>
      function changeText() {
        document.getElementById("title").innerText = "Text Changed!";
      }
    </script>
  </body>
</html>

2. Events

JavaScript can respond to user actions like clicks, mouse movements, and keyboard input.

Example:

document.getElementById("myButton").addEventListener("click", function () {
  alert("Button clicked!");
});

3. Asynchronous JavaScript

JavaScript can perform tasks asynchronously using setTimeout, Promises, and async/await.

Example:

// setTimeout
setTimeout(() => {
  console.log("This runs after 2 seconds.");
}, 2000);

// Promise
let promise = new Promise((resolve, reject) => {
  setTimeout(() => resolve("Done!"), 2000);
});
promise.then((result) => console.log(result)); // Done!

// async/await
async function fetchData() {
  let response = await fetch("https://api.example.com/data");
  let data = await response.json();
  console.log(data);
}

Advanced Concepts

1. Closures

A closure is a function that remembers its outer variables.

Example:

function outer() {
  let message = "Hello";
  function inner() {
    console.log(message);
  }
  return inner;
}
let myFunction = outer();
myFunction(); // Hello

2. Prototypes and Inheritance

JavaScript uses prototypes for inheritance.

Example:

function Person(name) {
  this.name = name;
}
Person.prototype.greet = function () {
  console.log("Hello, " + this.name);
};
let alice = new Person("Alice");
alice.greet(); // Hello, Alice

3. Modules

JavaScript modules allow you to split your code into separate files.

Example:

// math.js
export function add(a, b) {
  return a + b;
}

// main.js
import { add } from "./math.js";
console.log(add(2, 3)); // 5
