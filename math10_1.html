Relations and Functions - In-depth Overview

1. Relations

A relation is a connection or correspondence between elements of two sets. Formally, a relation from a set A to a set B is a subset of the Cartesian product A × B. This means that a relation consists of ordered pairs (a, b), where a ∈ A and b ∈ B.

Examples of Relations:

Set A = {1, 2, 3}, Set B = {a, b, c}: A relation R from A to B could be R = {(1, a), (2, b), (3, c)}, meaning that 1 relates to a, 2 relates to b, and 3 relates to c.

Types of Relations:

  • Reflexive Relation: A relation R on a set A is reflexive if every element in A is related to itself. In other words, ∀a ∈ A, (a, a) ∈ R.
    • Example: If A = {1, 2, 3}, a reflexive relation would include (1, 1), (2, 2), (3, 3).
  • Symmetric Relation: A relation R on set A is symmetric if for all a, b ∈ A, whenever (a, b) ∈ R, then (b, a) ∈ R.
    • Example: If R = {(1, 2), (2, 1)}, it is symmetric.
  • Antisymmetric Relation: A relation R is antisymmetric if for all a, b ∈ A, whenever (a, b) ∈ R and (b, a) ∈ R, then a = b.
    • Example: R = {(1, 2), (2, 3)} is antisymmetric because there is no case where both (a, b) and (b, a) are present with a ≠ b.
  • Transitive Relation: A relation R on set A is transitive if whenever (a, b) ∈ R and (b, c) ∈ R, then (a, c) ∈ R.
    • Example: If R = {(1, 2), (2, 3), (1, 3)}, the relation is transitive.

2. Functions

A function is a special type of relation where every element of the domain (set A) is related to exactly one element of the codomain (set B). The function can be written as f: A → B, where f(x) = y means that the element x in set A is mapped to the element y in set B.

Key Characteristics of Functions:

  • Domain: The set of all possible inputs (values from set A).
  • Range: The set of all possible outputs (values from set B).
  • Codomain: The set B, which contains all possible outputs of the function.

Types of Functions:

  • One-to-One (Injective): A function f is one-to-one if different elements of the domain map to different elements of the codomain. In other words, if f(a₁) = f(a₂), then a₁ = a₂.
  • Onto (Surjective): A function f is onto if every element of the codomain has at least one corresponding element in the domain. In other words, the range of the function is equal to its codomain.
  • One-to-One Correspondence (Bijective): A function is bijective if it is both one-to-one (injective) and onto (surjective). This means that there is a one-to-one correspondence between all elements of the domain and the codomain.

Notation:

f: A → B indicates that f is a function from set A to set B.

f(x) = y indicates that f maps x to y.

Examples of Functions:

  • f(x) = 2x + 3 is a function from the set of real numbers to the set of real numbers.
  • f(x) = x² is a function from real numbers to non-negative real numbers.

3. Important Formulas and Theorems

Function Composition:

If you have two functions f and g, the composition (f ∘ g)(x) = f(g(x)).

Example: If f(x) = 2x and g(x) = x + 1, then (f ∘ g)(x) = 2(x + 1) = 2x + 2.

Inverse Function:

If f is a function from A to B, the inverse function f⁻¹ is a function from B to A that satisfies:

f⁻¹(f(x)) = x and f(f⁻¹(y)) = y
Example: If f(x) = 2x + 3, the inverse function is f⁻¹(x) = (x - 3) / 2.

Domain and Range of Functions:

  • Domain: The set of all possible input values.
  • Range: The set of all possible output values.

Function Behavior:

  • Increasing Function: A function f(x) is increasing if f(x₁) < f(x₂) whenever x₁ < x₂.
  • Decreasing Function: A function f(x) is decreasing if f(x₁) > f(x₂) whenever x₁ < x₂.
  • Constant Function: A function f(x) is constant if f(x₁) = f(x₂) for all x₁, x₂ in the domain.

Slope of a Function (For Linear Functions):

The slope m of a linear function f(x) = mx + b is given by the change in y divided by the change in x:

m = Δy / Δx
